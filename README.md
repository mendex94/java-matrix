# Exercício de Matrizes

---

## Objetivo

Fazer um programa que leia um número inteiro `n` e gere uma matriz `n x n`. Em seguida, mostrar a diagonal principal e a quantidade de valores negativos da matriz.

## Input / Output

| Input                            	 | Output                                               	 |
|------------------------------------|--------------------------------------------------------|
| 3<br>5 -3 10<br>15 8 2<br>7 9 -4 	 | Main diagonal:<br>5 8 -4<br><br>Negative numbers = 2 	 |
